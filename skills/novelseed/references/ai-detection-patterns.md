# AI写作检测与去味指南

来源：snowflake-fiction humanize-text v2.0

## 24种AI痕迹检测模式

### 内容模式（5种）

| # | 模式 | 示例 | 修正 |
|---|------|------|------|
| 1 | **意义膨胀** | "标志着行业进入新阶段" | "这个政策2024年3月出台" |
| 2 | **浅层-ing分析** | "他深入探讨了这个问题" | "他把这事儿琢磨透了" |
| 3 | **模糊归因** | "有观点认为/研究表明" | "王教授2022年的论文发现" |
| 4 | **因果滥用** | "因为...所以...因此...从而..." | 省略因果词，让事实自然相连 |
| 5 | **过度总结** | "总而言之/综上所述/由此可见" | "所以说"或直接省略 |

### 语言模式（8种）

| # | 模式 | 说明 |
|---|------|------|
| 6 | **AI高频词汇** | 标志着、彰显、充分体现、不可或缺、赋能 |
| 7 | **三规则滥用** | "不仅...而且...同时..." → 最多一个递进 |
| 8 | **同义词轮替** | "审视/审阅/观察/观看" — AI怕重复，但真实写作应该有重复 |
| 9 | **破折号过多** | 每段超过1个破折号就该警觉 |
| 10 | **抽象上位词** | "情感纽带""价值体系""方法论框架" → 说具体 |
| 11 | **被动句式** | "被赋予了""被认为" → 主动语态 |
| 12 | **程度副词堆砌** | "非常""极其""极为""深刻" → 删掉或只用1个 |
| 13 | **排比用力过猛** | 排比超过3项=AI痕迹 |

### 风格模式（5种）

| # | 模式 | 说明 |
|---|------|------|
| 14 | **语气单一** | 全篇同一语气=AI，好人偶尔刻薄，坏人也说人话 |
| 15 | **无观点** | 只陈述不判断=AI，敢于说"我觉得""这不对" |
| 16 | **过度工整** | 段落长度一致、结构对称=AI |
| 17 | **缺乏节奏变化** | 长短句比例固定=AI，真实写作节奏是呼吸 |
| 18 | **情感抽离** | 上帝视角不投入=AI，写愤怒时文字应该愤怒 |

### 交流模式（3种）

| # | 模式 | 说明 |
|---|------|------|
| 19 | **过度礼貌** | "值得指出的是""值得注意的是" → 删 |
| 20 | **回避冲突** | AI不敢让角色真正吵架 → 让他们吵！ |
| 21 | **填充模糊** | "某种程度上""大体而言" → 要么说清楚，要么不说 |

### 进阶模式（3种）

| # | 模式 | 说明 |
|---|------|------|
| 22 | **比喻安全区** | "如虎添翼""如鱼得水"等成语比喻 → 换新鲜比喻 |
| 23 | **无身体感** | AI不会写饿/疼/累/冷/热 → 角色要有肉身 |
| 24 | **过度合理化** | 每个行动都解释动机 → 真实的人经常莫名其妙 |

## 去AI味四步法

### 步骤1：加入观点
AI不敢说"我觉得"。好文本一定有判断。
```
AI写：这个方案有多种可行性。
改写：这方案不行。第三个漏洞太大。
```

### 步骤2：变化节奏
```
AI写：(长句)(长句)(长句)(长句)
改写：(短句)(长句)(短句)。像这样。然后拉长，慢慢地说一段，再说一段，让句子有自己的呼吸。再短。
```

### 步骤3：承认复杂
```
AI写：这个问题可以通过以下方法解决。
改写：说实话我也不知道怎么解决。试过三个方案，两个失败了。剩下那个，只能说暂时还行。
```

### 步骤4：注入身体
```
AI写：他感到非常疲惫。
改写：眼睛干得像砂纸。肩膀那块肌肉跳了一整天。他想躺下来，什么姿势都行，只要不站着。
```

## 评分标准

| 评分 | 等级 | 说明 |
|------|------|------|
| 90+ | 人写的 | 找不到明显AI痕迹 |
| 75-89 | 自然 | 偶有生硬，整体流畅 |
| 60-74 | 明显AI味 | 多处模式匹配，需要改 |
| 50-59 | 重AI味 | 几乎每段都有问题 |
| <50 | 纯AI | 需要从头改写 |
