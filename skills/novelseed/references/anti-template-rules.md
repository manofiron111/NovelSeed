# 反模板与角色活人感

来源：novel-craft methods/ + patches/

## 四条铁律

### 铁律1：人物大于情节
> 模板化写作 = 情节推动人物，人物只是情节的棋子。
> 活人感写作 = 人物驱动情节，情节是人物选择的自然结果。

**自检**：如果把主角换成另一个人，情节是否完全不同？如果换成谁都可以，那就是模板化。

### 铁律2：冲突必须「不对称」
> AI 喜欢写势均力敌的对称冲突。真实的故事，冲突从来不对称。
> 一方以为自己赢了，另一方另有图谋。信息的错位就是冲突。

**自检**：双方是否都知道对方的底牌？如果都清楚，就是模板化。

### 铁律3：每章至少一个「不确定」
> 不确定 = 动机不确定 / 结果不确定 / 身份不确定 / 信息不确定。
> 没有不确定的章节 = 可以不写的章节。

### 铁律4：对话是微型战场
> 不是信息交换，是权力博弈。每句话都在进攻/防守/试探/退缩。

## 活人感注入（Liveness Patch）

### 角色要有：
1. **身体感觉** — 饿、疼、冷、累、痒、心跳
2. **不合理的情绪** — 该高兴时惆怅，该愤怒时笑
3. **口头禅与口癖** — 但必须自然，不能刻意重复
4. **固定的行为模式** — 紧张时咬手指、思考时转笔
5. **与他人不同的反应** — 同样事件，不同角色反应各异

### 角色不要：
1. ❌ 总是说「正确」的话
2. ❌ 永远理性决策
3. ❌ 情绪反应永远「恰到好处」
4. ❌ 说话风格与旁白风格一致
5. ❌ 所有角色用同样的比喻/修辞

### 反模板检查清单
| 模式 | 表现 | 必改 |
|------|------|:--:|
| "他的眼神深邃如..." | 每个角色都用同样的比喻结构 | ✓ |
| "她微微一笑" | 微笑是万能反应 | ✓ |
| "他沉默了一会儿" | 沉默是万能过渡 | ✓ |
| 反派必说狠话 | 反派只会放狠话=纸片人 | ✓ |
| 师父必说玄语 | "天机不可泄露"滥用 | ✓ |
| 女主必被救 | 女性角色缺乏主动性 | ✓ |

## 多人场景写作

来源：novel-craft multi-character.md

1. **谁在场？** — 列出所有在场角色
2. **谁主导？** — 这个场景由谁掌控话语权
3. **谁沉默？** — 沉默的人在想什么？（这往往比说话的人更重要）
4. **权力怎么流动？** — 谁进谁退，谁在这个场景中获得了/失去了什么
5. **每个人想要什么？** — 公开目标 vs 隐藏目标
